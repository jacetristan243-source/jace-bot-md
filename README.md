## ⚙️ Configuration

Modifiez `config.js`

| Variable | Description |
|---------|-------------|
| `global.owner` | Numéro propriétaire |
| `global.api.gemini` | Clé API Gemini |
| `global.api.tgToken` | Token Telegram |
| `global.adminTelegramIds` | ID admin Telegram |
| `global.mode` | public / private |
